# Roadmap

## Near-term (Next 3-6 months)

- [ ] **Additional S3 APIs**: Object versioning, tagging, policies ...
- [ ] **Performance**: Fractal ART performance optimizations
- [ ] **Availability**: Service HA for NSS
- [ ] **Scalability**: Support hundreds of NSS and BSS instances
- [ ] **Durability**: Replication for instance failures. Enhanced data protection mechanisms (e.g Deterministic simulation testing)
- [ ] **Monitoring**: Enhanced metrics and dashboards, Cloudtrail etc.

## Medium-term (6-12 months)

- [ ] **Erasure Coding**: Space-efficient data protection
- [ ] **Server-side Encryption**: SSE-S3 and SSE-KMS support
- [ ] **Lifecycle Policies**: Automated object expiration and transitions
- [ ] **Access Controls**: Bucket policies and IAM integration
- [ ] **Cross-region Replication**: Multi-region data synchronization
- [ ] **Compression**: Transparent object compression
- [ ] **Analytics**: Storage analytics and insights
- [ ] **Multi-AZ High Availability**: Complete multi-AZ data replication and failover
- [ ] **Multi-cloud**: Support for other cloud providers
